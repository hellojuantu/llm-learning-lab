# Training Loop Project

角色：`pro`

目标：闭合 forward、loss、zero_grad、backward、update 的最小训练循环。

本节文件：

```text
preview.ipynb    上课前先做，填 TODO 后跑 qa_check
class.ipynb      上课实验，先运行代码再拆概念
homework.ipynb   正式作业，编号 TODO、qa_check、Debug Lab
review_prompt.md 学完后用来复盘
```

学习顺序：

```text
preview.ipynb -> class.ipynb -> homework.ipynb -> review_prompt.md
```

完成标准：把 `homework.ipynb` 里的 TODO、测试、Debug Lab 跑通，并能用自己的话解释本节目标。
